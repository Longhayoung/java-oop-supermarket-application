# Java OOP Supermarket Application

## Overview
This project is a Java-based object-oriented application that simulates a simple
supermarket checkout system. The application demonstrates core OOP principles
through a structured and extensible design.

## Features
- Abstract product model with polymorphic behaviour
- Separate product types (Fresh Fruit and Packaged Items)
- Tax calculation based on product type
- File-based product data loading
- Interactive product selection and receipt generation

## OOP Concepts Applied
- Inheritance and method overriding
- Polymorphism through a common product interface
- Encapsulation of product data and behaviour
- Separation of concerns across classes

## Technologies
- Java
- Object-Oriented Programming
- File I/O
